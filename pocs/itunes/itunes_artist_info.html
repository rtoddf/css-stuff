Searching for an Artist - More Info

The javascript will construct a search string with that artist and return the first eight tracks from that artist - if it finds any results. Type in an artist's name into the search box and click Submit.

  • artistName - The artist or group's name.
  • artistId - The id that can be used for other searches, such as albums.
  • artistViewUrl - The url for a bio and list of media by that artist.
  • trackName - The track name of the song the search returned.
  • trackId - The id of the above track that can be used in other searched.
  • releaseDate - The unicode formatted date of the release of the track above. (Some extra work has been done to extract just the year)
  • kind - The media kind of the track above. You can use this to narrow down your search.
  • trackPrice - The price of the track above.
  • trackTimeMillis - The length (in milliseconds) of the track above.
  • previewUrl - the url of a preview of the track above.
  • artworkUrl100 - the url of the artwork for that track/artist. We'll use this later in an example to get other sizes.
  • primaryGenreName - The primary genre of the artist.
